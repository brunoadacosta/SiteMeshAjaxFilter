    <font size="2" face="Verdana, Arial, Helvetica, sans-serif">
<i>The Open Source J2EE component project</i>
<p>OpenSymphony is an Open Source project dedicated to providing enterprise class
  J2EE applications and components. Our components aim to be specification compliant,
  and should work in any J2EE compliant application server. (Although some components
  only require a servlet container rather than a full J2EE app server).</p>
<p>See <a href="whatis.html;jsessionid=BCEAHBFFKKMD">What Is OpenSymphony?</a> and the OpenSymphony <a href="architecture.html;jsessionid=BCEAHBFFKKMD">Architectural
  Overview</a> <font size="2" face="Verdana, Arial, Helvetica, sans-serif">for
  more detail.</font></p>
</font>

    <font size="2" face="Verdana, Arial, Helvetica, sans-serif">
      <h3><a href="components.html;jsessionid=BCEAHBFFKKMD">Components</a></h3>
	  <h4>Production Quality</h4>
		<ul>
		  <li><b><a href="oscache/;jsessionid=BCEAHBFFKKMD">OSCache</a></b> - A JSP tag library and set of
          classes to perform fine grained dynamic caching of JSP content. It also
          has persistent on disk caches, and can allow your site to have graceful
          error tolerance (eg if an error occurs like your db goes down, you can
          serve the cached content so people can still surf the site almost without
          knowing). [requires Servlet 2.2 / JSP 1.1] </li>
		<li><b><a href="sitemesh/;jsessionid=BCEAHBFFKKMD">SiteMesh</a></b> - This is a servlet filter
          + tag library component that uses decorators to allow for excellent
          site layout and web app integration (ie you can integrate legacy web
          apps that are perl or CF based with your current J2EE apps - very cool)
          [requires Servlet 2.3 / JSP 1.1] </li>
		<li><b><a href="formtags/;jsessionid=BCEAHBFFKKMD">FormTags</a></b> -
          A JSP taglib designed to make integration of HTML forms and Javabeans
          relatively orthogonal and convenient. </li>
        <li><b><a href="transformtags/;jsessionid=BCEAHBFFKKMD">Transformtags</a></b>
          - A JSP taglib designed to provide tags to transform source content
          from one form to another. For example, it contains a tag to apply XSL
          to XML content, as well as a tag to escape HTML text and search/replace
          content.</li>
        <li><b><a href="oscore/;jsessionid=BCEAHBFFKKMD">OSCore</a></b> - Core components that are useful
          in any J2EE application. The most used component here is the PropertySet
          EJB, an EJB properties object that's very flexible and can be constrained
          by a PropertySetSchema.</li>
	  </ul>
	  <h4>Beta (Developing Components)</h4>
      <ul>

        <li><b><a href="osuser/;jsessionid=BCEAHBFFKKMD">OSUser</a></b> - A set of user management EJB
          components that will work on any EJB server. At the moment every server
          has its own user management API. This project is an attempt to unite
          them to make your applications <i>truly</i> portable between app servers.
          [requires EJB 1.1 / OSCore components] </li>
        <li><b><a href="richtags/;jsessionid=BCEAHBFFKKMD">RichTags</a></b> -
		  A JSP taglib designed to allow rich GUI user-interface and data-entry
		  widgets to be used in HTML based web-applications.
        <li><b><a href="webcompass/;jsessionid=BCEAHBFFKKMD">WebCompass</a></b> - A combination
          of an EJB package and a tag library to provide site navigation and organization.
        </li>
        <li><b><a href="osworkflow/;jsessionid=BCEAHBFFKKMD">OSWorkflow</a></b> - A very flexible workflow system that can be plugged in to
		almost any need or existing application. Uses can range from document management
		to order processing to project management. [requires EJB 1.1 / OSCore & OSUser components] </li>
      </ul>

    <font size="2" face="Verdana, Arial, Helvetica, sans-serif">
<i>The Open Source J2EE component project</i>
<p>OpenSymphony is an Open Source project dedicated to providing enterprise class
  J2EE applications and components. Our components aim to be specification compliant,
  and should work in any J2EE compliant application server. (Although some components
  only require a servlet container rather than a full J2EE app server).</p>
<p>See <a href="whatis.html;jsessionid=BCEAHBFFKKMD">What Is OpenSymphony?</a> and the OpenSymphony <a href="architecture.html;jsessionid=BCEAHBFFKKMD">Architectural
  Overview</a> <font size="2" face="Verdana, Arial, Helvetica, sans-serif">for
  more detail.</font></p>
</font>

    <font size="2" face="Verdana, Arial, Helvetica, sans-serif">
      <h3><a href="components.html;jsessionid=BCEAHBFFKKMD">Components</a></h3>
	  <h4>Production Quality</h4>
		<ul>
		  <li><b><a href="oscache/;jsessionid=BCEAHBFFKKMD">OSCache</a></b> - A JSP tag library and set of
          classes to perform fine grained dynamic caching of JSP content. It also
          has persistent on disk caches, and can allow your site to have graceful
          error tolerance (eg if an error occurs like your db goes down, you can
          serve the cached content so people can still surf the site almost without
          knowing). [requires Servlet 2.2 / JSP 1.1] </li>
		<li><b><a href="sitemesh/;jsessionid=BCEAHBFFKKMD">SiteMesh</a></b> - This is a servlet filter
          + tag library component that uses decorators to allow for excellent
          site layout and web app integration (ie you can integrate legacy web
          apps that are perl or CF based with your current J2EE apps - very cool)
          [requires Servlet 2.3 / JSP 1.1] </li>
		<li><b><a href="formtags/;jsessionid=BCEAHBFFKKMD">FormTags</a></b> -
          A JSP taglib designed to make integration of HTML forms and Javabeans
          relatively orthogonal and convenient. </li>
        <li><b><a href="transformtags/;jsessionid=BCEAHBFFKKMD">Transformtags</a></b>
          - A JSP taglib designed to provide tags to transform source content
          from one form to another. For example, it contains a tag to apply XSL
          to XML content, as well as a tag to escape HTML text and search/replace
          content.</li>
        <li><b><a href="oscore/;jsessionid=BCEAHBFFKKMD">OSCore</a></b> - Core components that are useful
          in any J2EE application. The most used component here is the PropertySet
          EJB, an EJB properties object that's very flexible and can be constrained
          by a PropertySetSchema.</li>
	  </ul>
	  <h4>Beta (Developing Components)</h4>
      <ul>

        <li><b><a href="osuser/;jsessionid=BCEAHBFFKKMD">OSUser</a></b> - A set of user management EJB
          components that will work on any EJB server. At the moment every server
          has its own user management API. This project is an attempt to unite
          them to make your applications <i>truly</i> portable between app servers.

        <li><b><a href="richtags/;jsessionid=BCEAHBFFKKMD">RichTags</a></b> -

		  widgets to be used in HTML based web-applications.
        <li><b><a href="webcompass/;jsessionid=BCEAHBFFKKMD">WebCompass</a></b> - A combination
          of an EJB package and a tag library to provide site navigation and organization.
        </li>
        <li><b><a href="osworkflow/;jsessionid=BCEAHBFFKKMD">OSWorkflow</a></b> - A very flexible workflow system that can be plugged in to
